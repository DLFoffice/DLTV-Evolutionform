# ชื่อโครงการ (คลิกบรรทัดเพื่อดูรายละเอียด) ยุทธศาสตร์ งบอนุมัติ (บาท) ใช้ไป (บาท) PO (บาท) ใช้+PO (บาท) คงเหลือสุทธิ สถานะ ตัวชี้วัด จัดการ
📊 เลือกโครงการเพื่อรายงานผลการดำเนินงาน
เลือกโครงการที่กรอกเข้าระบบไว้แล้วจากรายการด้านล่าง (จัดกลุ่มตามยุทธศาสตร์) เพื่อให้ข้อมูลที่รายงานถูกผูกกับโครงการและหมวดหมู่ยุทธศาสตร์ที่ถูกต้องเสมอ หรือคลิกเลือกโครงการจากตารางด้านล่างได้โดยตรง
🎯 ตัวชี้วัดของโครงการ (KPI) — ผลที่ทำได้จริง
ระบุผลการดำเนินงานเทียบกับตัวชี้วัดของโครงการด้วยตนเอง (กด Enter หรือปุ่ม + เพื่อเพิ่มรายการ)

ส่งข้อมูลโครงการทั้งหมดที่มีอยู่ขึ้น Google Sheet ทันที (เขียนทับข้อมูลเดิม)
ส่งออกข้อมูล (CSV)

รูปแบบข้อมูลที่รองรับ
Sheet ควรมีหัวคอลัมน์แถวแรกดังนี้:

id | name | strategy | budget | spent | po | status | result | kpi | problems | solutions | quarter | owner | committees | image_count | images_json | lastEditedBy | lastEditedByPosition | lastEditedAt

• strategy = 1, 2, 3, 4 หรือ 5 (งบบริหารสำนักงาน)
• budget, spent, po = จำนวนเงิน (ตัวเลข ไม่ใส่ comma)
• status = done / progress / pending
• kpi = ตัวชี้วัด คั่นด้วย | (pipe)
• quarter = 1, 2, 3, 4 หรือ all
• owner = ผู้รับผิดชอบ / หน่วยงาน
• committees = JSON array เช่น ["นโยบายและแผนงาน","วิชาการ"]
• images_json = JSON array ของรูปภาพ [{"name":"...","url":"...","publicId":"..."}] (Cloudinary URL ไม่ใช่ base64)
• lastEditedBy = ชื่อผู้แก้ไขล่าสุด (ดึงจากบัญชีที่ login อัตโนมัติ) — คอลัมน์ใหม่
• lastEditedByPosition = ตำแหน่งของผู้แก้ไขล่าสุด — คอลัมน์ใหม่
• lastEditedAt = วันเวลาที่แก้ไขล่าสุด (ข้อความ) — คอลัมน์ใหม่
